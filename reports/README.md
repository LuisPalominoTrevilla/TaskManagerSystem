## Service Registry #

The service registry is responsible for registering microservices and act as a gatweay API.

## Requirements

- [Node version 10.14](https://nodejs.org/en/) or higher is required. You can check it with the following command:

    `node -v`
- Npm version  6.5 is required, it comes with node. Check it with:

    `npm -v`

- Redis is required. Download it [here](https://redis.io/download)

## Run

- Install dependencies with:

    `npm install`
- Start Redis server on a separate command line window:

    `redis-server`
- Run the app:

    `npm start`
